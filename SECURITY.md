# Security Policy

This repository contains AI agent skills for biomedical research. It does not process or store patient data.

## Reporting Vulnerabilities

If you find credentials, personal information, or patient data in this repository:

1. Do NOT open a public issue
2. Open a private security advisory via GitHub: Settings > Security > Advisories
3. We will remove the content within 24 hours

## Dataset Policy

All datasets referenced in this repository must be publicly available from established repositories (TCGA, GEO, CellxGene, GDSC, DepMap) and referenced by accession number, not local file path.
